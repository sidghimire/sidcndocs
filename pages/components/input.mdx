## Add Input


```jsx copy
npx sidcn add input
```

## Import Input


```jsx copy
import { Input } from '/components/sidcn/ui';
```
### How to use

You can have Icon | Image | View | Text | Svg Component for Icon Purpose, Alert Title and Alert Description

```jsx copy
<Input type="Password" placeholder="Password" />
```

`Use with className`

```jsx copy
<View className="w-full">
    <Label>Hi</Label>
    <Input type="Password" placeholder="Password" className="p-4" />
</View>
```
