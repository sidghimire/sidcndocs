# Collapsible Component

The `Collapsible` component in **sidcn** provides an easy way to create collapsible sections, allowing you to organize and present information with flexibility. With components like `CollapsibleTrigger` and `CollapsibleContent`, you can build interactive and space-efficient layouts for your React Native and Expo applications.

## Import Collapsible

Begin using the `Collapsible` component by importing it from **sidcn**.

```jsx copy
import { Collapsible, CollapsibleTrigger, CollapsibleContent } from 'sidcn';
```
### How to use

Create a collapsible section with a trigger and content.
```jsx copy
<Collapsible>
  <CollapsibleTrigger>
    Can I use this in my project?
  </CollapsibleTrigger>
  <CollapsibleContent>
    Yes. Free to use for personal and commercial projects. No attribution required.
  </CollapsibleContent>
</Collapsible>

```

`Use with className`

Apply custom styling to the Collapsible component using Tailwind CSS classes or custom styles.

```jsx copy
<Collapsible className="w-[350px] space-y-2">
  <View className="flex flex-row items-center justify-between space-x-4 px-4">
    <Text className="text-md font-semibold">
      @sidghimire starred 3 repositories
    </Text>
    <CollapsibleTrigger>
      <View>
        <ChevronsUpDownIcon color={'#000'} size={20} />
      </View>
    </CollapsibleTrigger>
  </View>
  <View className="rounded-md border border-gray-400 px-4 py-3 text-sm mb-2">
    <Text>@sidcn/docs</Text>
  </View>
  <CollapsibleContent className='space-y-2'>
    <View className="rounded-md border border-gray-400 px-4 py-3 text-sm">
      <Text>@sidcn/ui</Text>
    </View>
    <View className="rounded-md border border-gray-400 px-4 py-3 text-sm">
      <Text>@sidcn/docs</Text>
    </View>
  </CollapsibleContent>
</Collapsible>
```
