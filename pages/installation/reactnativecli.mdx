# Installing sidcn for React Native CLI

## 1. Create the project

```bash
npx react-native init AwesomeProject
cd AwesomeProject
```
## 2. Install sidcn

```bash copy
yarn add sidcn
```

## 3. Initialize sidcn

```bash copy
npx sidcn init
```

## 4. Reset Cache

```bash copy
yarn start --clear // yarn

npm start -- --reset-cache // npm
```
